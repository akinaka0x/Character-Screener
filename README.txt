得られた文字画像データの仕分けをするためのアプリです。
kakezan、warizanでそれぞれ独立しています。
ここではkakezanについて説明しますが、warizanについても全く同様です。


kakezanディレクトリに予め用意したimages.npzをコピーしておいてください。
コマンドプロンプトでkakezanディレクトリに移動して
jupyter notebook
と打つとブラウザが立ち上がります。
そして、main.ipynbを選んでください。
セルを選択した状態で、Ctrl + Enterを押すと各セルを実行できます。

文字画像が現れるので問題がなければそのままエンターを、
問題があれば何が文字を打ってからエンターを押してください。
問題がない画像はoutputに、問題のある画像はignoredにナンバリングされて追加されます。
最後のデータまで行くとお疲れ様！と表示されます。

一つのデータセットの確認が終わったらimages.npzを交換して同じ作業を続けてください。
